    CIFAR10CNN is a Convolutional Neural Network trained on the CIFAR10 dataset. CIFAR10CNN is trained to recognize the difference between the 10 categories:

    • Airplane
    • Car/Automobile
    • Bird
    • Cat
    • Deer
    • Dog
    • Frog
    • Horse
    • Ship
    • Semitruck

    Choose a file containing one of these objects and submit to continue. You can also download the model used, from the link at end of the page. Firstly choose am image you want to test by clicking on "Choose File" and then click on "Submit" button besides Choose file option. Please upload an image in the following formats only PNG, JPG, and JPEG.
